## 🎯 About the Project

<table>
<tr>
<td>

**LuxeStay — Hotel Room Booking Management System** is a complete full-stack web application that digitizes hotel operations. It provides a powerful admin dashboard to manage rooms, guests, bookings, and availability — all from a single interface.

Built on the **CodeIgniter PHP framework** with a clean **MySQL** database schema, it's lightweight, fast, and easy to deploy on any **XAMPP** / **LAMP** / **WAMP** server.

### 🎯 Perfect For:
- 🧑‍💻 **Developers** learning PHP + CodeIgniter
- 🏫 **Students** submitting final year projects
- 🏨 **Small hotels** needing a simple booking solution
- 🔧 **Contributors** wanting to build on a real-world system

## 🚀 Getting Started

### 📋 Prerequisites

Make sure you have the following installed:

```
✅ XAMPP  (Apache + MySQL + PHP)    → https://www.apachefriends.org
✅ PHP >= 7.4
✅ MySQL >= 5.7
✅ Git                              → https://git-scm.com
✅ Any modern browser (Chrome/Firefox)
```

<br/>

### 📥 Step 1 — Clone the Repository

```bash
git clone https://github.com/karthikbm33/Hotel-Room-Booking-Management-System.git
```

<br/>

### 📁 Step 2 — Move to XAMPP htdocs

```bash
# Windows
move Hotel-Room-Booking-Management-System C:\xampp\htdocs\

# macOS
mv Hotel-Room-Booking-Management-System /Applications/XAMPP/htdocs/

# Linux
mv Hotel-Room-Booking-Management-System /opt/lampp/htdocs/
```

<br/>

### 🗄️ Step 3 — Import the Database

```
1. Open XAMPP Control Panel
2. Start  →  Apache  ✅
3. Start  →  MySQL   ✅
4. Open browser → http://localhost/phpmyadmin
5. Click "New" → Create database → Name it: lodge
6. Click "Import" tab
7. Choose file → select lodge.sql from the repo root
8. Click "Go" → Database imported ✅
```

<br/>

### ⚙️ Step 4 — Configure Database

Open this file:
```
application/config/database.php
```

Update with your settings:
```php
$db['default'] = array(
    'dsn'      => '',
    'hostname' => 'localhost',    // ← your host
    'username' => 'root',         // ← your MySQL username
    'password' => '',             // ← your MySQL password (blank for XAMPP default)
    'database' => 'lodge',        // ← must match your DB name
    'dbdriver' => 'mysqli',
    'dbprefix' => '',
    'pconnect' => FALSE,
    'db_debug' => TRUE,
    'cache_on' => FALSE,
);
```

<br/>

### ▶️ Step 5 — Run the Project

```
Open browser → http://localhost/Hotel-Room-Booking-Management-System
```

🎉 **You're live!**

## 🗃️ Database Setup

The `lodge.sql` file contains the complete database schema. Here's what's included:

```sql
-- Tables included in lodge.sql
✅ users          → Admin login accounts
✅ rooms          → Room details (type, price, status)
✅ bookings       → Reservation records
✅ guests         → Guest information
✅ payments       → Payment tracking
```

**To reset the database:**
```bash
1. Go to phpMyAdmin → Select 'lodge' database
2. Click Operations → Drop the database
3. Re-create 'lodge' and re-import lodge.sql
```

## ⚙️ Configuration

### Base URL Setup
```php
// application/config/config.php
$config['base_url'] = 'http://localhost/Hotel-Room-Booking-Management-System/';
```

### .htaccess (Remove index.php from URL)
```apache
RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ index.php/$1 [L]
```
